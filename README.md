## 🚀 Projetos Destaque

### 🎯 **SBTB - SimuTrader Binance Bot**
**Um bot de trading inteligente e adaptativo para Binance com dual-strategy**

[![GitHub](https://img.shields.io/badge/view%20repo-181717?style=for-the-badge&logo=github)](https://github.com/Pissolato32/SBTB)
[![Status](https://img.shields.io/badge/status-Production%20Ready-brightgreen?style=for-the-badge)](https://github.com/Pissolato32/SBTB)

**Highlights:**
- 🧠 **Dual Strategy Mode:** SCALP (1.5% TP) | SWING (5% TP) | AUTO (Volatility Detection)
- 📊 **Historical Analysis:** 7 dias de dados OHLCV com SMA200 + RSI intelligent entry
- 🛡️ **Enterprise Ready:** Prisma ORM, async-mutex thread-safety, WebSocket live dashboard
- 🧪 **90%+ Test Coverage:** Vitest + CI/CD GitHub Actions
- 🐳 **Docker Support:** Production containerization + Render deployment

**Stack:** React 19 | Vite | Node.js | Prisma | PostgreSQL | CCXT | TensorFlow.js

### 🛡️ **VeritasAI - Fact-Checking com IA**
**Plataforma de verificação de fatos agnóstica com múltiplos modelos de IA**

[![GitHub](https://img.shields.io/badge/view%20repo-181717?style=for-the-badge&logo=github)](https://github.com/Pissolato32/VeritasAI)
[![Status](https://img.shields.io/badge/status-Alpha-blue?style=for-the-badge)](https://github.com/Pissolato32/VeritasAI)
